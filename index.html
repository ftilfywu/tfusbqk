О проекте

Нейросиноптик

Программа, которая даёт точные
прогнозы погоды

Я написала её на языке Python,
используя библиотеку Pyweather.
Общая длина кода измеряется
сотнями строк.

На картинке справа можно
увидеть пример кода и работы
программы

Как это работает?

  1. Пользователь делает запрос

    Для этого мы создали
    удобный интерфейс
  2. Программа использует
    специальные библиотеки

    PyWeather - библиотека,
    которая даёт доступ к данным
    о погоде
  3. Запрашивает данные
    с других серверов

    Программа запращивает
    данные с сайта, где
    профессионально
    занимаются прогрнозами
  4. Преобразует запрос в
    голос и озвучивает его

    Для преобразования текста
    в голос программа
    использует библиотеку
    SpeakerPy
